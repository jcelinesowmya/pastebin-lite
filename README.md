# Pastebin Lite

A simple Pastebin-like web application where users can create a text paste
and share a link to view it.

## Problem Statement
Build a small Pastebin-like application where:
- A user can create a text paste
- A unique link is generated
- Anyone with the link can view the paste

## Features
- Create text pastes
- Generate shareable links
- View pastes using unique IDs
- Clean and responsive UI

## Tech Stack
- Next.js (App Router)
- React
- TypeScript
- CSS

## How to Run the Project Locally

```bash
npm install
npm run dev
Open your browser and go to: http://localhost:3000

How It Works

User enters text and submits

A unique paste ID is generated

The paste can be accessed at /p/[id]

Data is stored in memory (resets on server restart)"# pastebin-lite"
